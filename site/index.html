LIVE ROBLOX STUDIO INSPECTOR

Studio Stud

An AI-first, read-only live picture of a Roblox place's DataModel. A local Rust daemon + one Studio plugin, compact JSON for agents, policy-gated writes, and a Rojo-style project diff. Windows-only, localhost-only, never mutates the place.

Install into any repo

irm https://tyleradams2002.github.io/studio-stud/install.ps1 | iex

With the AI workflow rule + command:

& ([scriptblock]::Create((irm https://tyleradams2002.github.io/studio-stud/install.ps1))) -WithCursorRule

Then

.\studio-stud doctor
.\studio-stud serve      # leave running
.\studio-stud capture
.\studio-stud analyze
